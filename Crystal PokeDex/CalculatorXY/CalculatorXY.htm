程序信息 ±

如何下載計算器

在瀏覽器窗口菜單中選擇 文件 -> 另存為... 將網頁保存到您的計算機中（請務必選擇“網頁，僅 HTML”方式保存）。該程序在脫機狀態下仍然可用。

兼容性：

由於神奇寶貝X‧Y中一些神奇寶貝的種族值有變化，這個計算器並不適合之前的遊戲，可以使用 適合第五世代及之前版本的個體值計算器。

關於

個體值 / 能力值 / 催醒計算器由 Legendary Pokémon 管理員 Hercules G. Papatheodorou (Arty2) 開發。該程序可以通過神奇寶貝的種族、性格、能力和基礎得點數據來計算個體值。

該程序由 JavaScript 編寫，具有跨操作系統的兼容性，只需一個文件和具有 JavaScript 功能的瀏覽器即可。該程序 應該可以 在任何操作系統的 Firefox, Chrome, Safari, Opera, Internet Explorer 等瀏覽器運行（任天堂 DSi 和 3DS 由於字庫問題不支持中文版）。

個體值計算（簡潔模式？）

* 星號標記的為可選項。

注意一定要記錄基礎得點，必要時可以使用降低基礎得點樹果，或使用超級訓練系統的重置袋清除所有基礎得點。

能力判斷師位於奇南市神奇寶貝中心，可以告訴你神奇寶貝的全體能力和最高個體值評價，以及個體值為0的能力。在計算器中，可以使用 Ctrl/CMD 鍵選擇（取消選擇）多項能力。

使用神奇糖果將神奇寶貝提升到等級 20，就可以得到 100% 準確的能力值，在使用之前記得存檔。

神奇寶貝模式：神奇寶貝的等級越高，計算出的個體值越準確。如果在較低的等級計算，請盡量多填寫可選的內容。+等級 和 +基礎得點 是指要計算出更精確的結果所需要的等級或基礎得點。不同型態或百萬進化的能力值也可以幫助計算。

隊伍模式：可以一次計算多個神奇寶貝的個體值。

單行模式：可以計算多個神奇寶貝的個體並分別查看。

能力值計算

使用平均個體值，基礎得點來準確計算出特定等級下神奇寶貝的能力值。

催醒計算

催醒的屬性和威力取決於神奇寶貝的個體值，如果已知屬性，可以更容易計算。

催醒超能力者位於百刻市神奇寶貝中心以東屋子里。

使用個體值分布（或平均個體值）可以得到所有可能的組合。

基礎得點訓練

基礎得點追蹤
取消選擇選中的一行可以進入追蹤模式，選擇相應的道具、對手的神奇寶貝和數量來計算得到的基礎得點。
基礎得點計算
首先計算出神奇寶貝的能力值，然後改變第一行中的能力值，然後得出到達特定的能力值尚需的基礎得點。
